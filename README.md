# CSS-COOKBOOK

[中文说明](./README-ZH.md)

## Install

### Clone
> git clone git@github.com:GreenMelon/css-cookbook.git

### Node Modules
> npm install

### Run
> npm run dev

> open localhost:8181 in browser
